# 樱花动漫、风车动漫弹幕播放

自动匹配加载动漫剧集对应弹幕并播放，目前支持樱花动漫、风车动漫

### 支持网站

[樱花动漫、风车动漫](https://www.dmla4.com/)

### 主要功能

- 自动加载匹配的在线弹幕
- 可手动上传 xml 格式本地弹幕

### 主要原理

根据页面获取动漫名称，
通过 url 确定播放剧集，
重新渲染播放器，
